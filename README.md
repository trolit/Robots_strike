#            Robots Strike  		#
wieloosobowa strzelanka 3D wykonana na silniku Unity.
Do dyspozycji gracza są 3 różne klasy: Zwiadowca,
Snajper i Żołnierz - każda z nich ma zróżnicowane
statystyki i inną broń. Podziękowania dla Asbjorn'a
Thirslund'a.
